# How to Use

### 目的

すでにjenkinsが作成されているもの。

### repositoryの取得

```
$ git clone git@github.com:iganari/base-jenkins-dev.git
```

### vagrant立ち上げ

```
$ cd opsfiles/vagrant
$ vagrant up
```

### ブラウザでjenkinsにアクセス
